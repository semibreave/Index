Apache Installation Steps

  1. Get the latest Apache Lounge Binary:For Windows,installation must be done from one of many binaries. Lounge is one of them. Visit Apache Lounge website https://www.apachelounge.com/download/.
  2. Download the .zip file: The top of the link is likely the latest.
  3. Extract the .zip file: The result would be a folder named Apache24. I would normally extract it to C:/ .
  4. Install Apache: Open up CMD. Navigate to Apache24\bin. Run 'httpd -k install'.
  5. Start Apache: Run services.msc. Locate the service name Apache24 and start it.
  6. Test Apache: Try access localhost on browser to see if the web service is running."It works" should be displayed rendered from index.html.

PHP Installation Steps

  1. Get the latest PHP for Windows:As of this is written,the URL is https://windows.php.net/download#php-8.2
  2. Download the .zip file:Choose the thread safe version from the list .
  3. Extract the .zip file: Make a folder before extracting to avoid clutter.The rest of the steps assuming the .zip file is extracted to C:/PHP.
  4. Configure Apache to handle .php file type: Edit the file Apache24/httpd.conf. Add the following lines(This applied to php 8 at the time of writing):
    • LoadModule php_module "C:/PHP/php8apache2_4.dll"
    • AddType application/x-httpd-php .php
    • LoadFile "C:/PHP/php8ts.dll"
    • PHPIniDir "C:/PHP"
  5. Restart Apache service:Run services.msc.Locate the service name Apache24 and restart it.
